# OpenAI Prompt 使用示例

## 方式 1：直接在 ChatGPT Web 界面使用

1. 打开 ChatGPT（https://chat.openai.com）
2. 复制下面的 SYSTEM 部分，在对话框中输入："请扮演以下角色："
3. 粘贴 SYSTEM 提示
4. 复制 USER 部分，将 `{answers}` 替换为实际的 JSON 数据
5. 发送消息并等待回复

---

## 方式 2：替换示例

### 原始 Prompt（需要替换 {answers}）：

```
SYSTEM:
你是资深商业咨询师 + 产品经理...

USER:
以下是用户提交的答案（JSON）：
{answers}

请基于这些信息，生成一份判断报告...
```

## 方式 3：通过 OpenAI API 使用

参考 `gpt-service.js` 文件中的 `generateReportWithOpenAI` 函数。

### 代码示例：

```javascript
const { OpenAI } = require('openai');
const fs = require('fs');

const openai = new OpenAI({
  apiKey: process.env.OPENAI_API_KEY
});

async function generateReport(answers) {
  const systemPrompt = fs.readFileSync('./prompts/openai_prompt.txt', 'utf-8')
    .split('USER:')[0]
    .replace('SYSTEM:', '')
    .trim();
  
  const userPrompt = fs.readFileSync('./prompts/openai_prompt.txt', 'utf-8')
    .split('USER:')[1]
    .replace('{answers}', JSON.stringify(answers, null, 2));

  const response = await openai.chat.completions.create({
    model: 'gpt-4',
    messages: [
      { role: 'system', content: systemPrompt },
      { role: 'user', content: userPrompt }
    ],
    temperature: 0.7,
    max_tokens: 2000
  });

  return response.choices[0].message.content;
}

// 使用
const answers = {
  q1: "我想开发一个健身类的 SaaS 产品...",
  // ... 其他问题
};

const report = await generateReport(answers);
console.log(report);
```

## 注意事项

1. **API 密钥安全**：不要将 API 密钥提交到代码仓库
2. **费用控制**：每次调用都会产生费用，建议设置使用限制
3. **响应时间**：GPT-4 响应可能需要几秒钟，请设置合适的超时时间
4. **错误处理**：API 可能失败，需要做好错误处理和重试机制
5. **格式一致性**：确保输出格式与预期一致，可能需要后处理

## 替代模型

如果不想使用 GPT-4，可以考虑：

- **GPT-3.5-turbo**：成本更低，速度更快，但质量稍差
- **Claude**（Anthropic）：质量接近 GPT-4
- **文心一言**（百度）：国内可访问
- **通义千问**（阿里云）：国内可访问

只需修改 API 调用部分即可。
